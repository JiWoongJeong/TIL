> **[출력]**
>
> 각 줄마다 "#T" (T는 테스트 케이스 번호)를 출력한 뒤, 답을 출력한다.
>
> #1 4 
>
> #2 32402 
>
> #3 13398

- 풀이

```python
T = int(input())
for tc in range(1, T + 1):
    N, M, L = list(map(int, input().split()))
    nums = list(map(int, input().split()))
    for _ in range(M):
        i, j = list(map(int, input().split()))
        nums.insert(i, j)

    print("#{0} {1}".format(tc, nums[L]))
```
